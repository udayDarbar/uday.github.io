# Udaysinh Rathod — Research Portfolio

A clean, modern academic portfolio website showcasing research in machine learning, computer vision, geospatial AI, and healthcare informatics.

## 🎯 Design Principles

- **Research-first**: Content prioritized for PhD committees, collaborators, and faculty
- **Minimalist aesthetic**: No unnecessary visuals or marketing language
- **Accessibility**: Semantic HTML, WCAG-compliant contrast, mobile-responsive
- **GitHub Pages ready**: No build step required; works on static hosting

## 📂 Structure

```
uday.github.io/
├── index.html                 # Main portfolio page
├── assets/
│   ├── Uday_CV.pdf           # Curriculum vitae
│   └── images/               # Project diagrams
├── css/                      # Optional modular styling
└── js/                       # Minimal JavaScript
```

## 🎨 Key Features

### ✅ Publications Organization
- **Journal Articles**: Peer-reviewed journals with DOI links
- **Conference Papers**: IEEE, ACM, and specialized venues
- **Workshops & Posters**: Clearly labeled as such
- **Accepted & Under Review**: Transparent status with badges

### ✅ Projects Section
Each project includes:
- Title and technical description
- SVG-based conceptual diagrams (not stock photos)
- Tools and methods used
- 2–3 impact-focused bullet points
- Publication or status badge

### ✅ Responsive Design
- Desktop: 2-column grid for projects
- Tablet: Single column layout
- Mobile: Optimized typography
- Print-friendly: Clean PDF export

- ✅ External links open in new tab with security markers

## 🚀 Quick Deployment

Push to GitHub repository `username.github.io`:

```bash
git commit -m "Update portfolio"
git push origin main
```

Site automatically deploys to `https://username.github.io`
